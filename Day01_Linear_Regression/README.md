## Definition
Linear Regression is a supervised learning algorithm used to
predict a continuous dependent variable based on input features.
It makes the best fit line between the one independent variable and one dependent variable.

## Formula
y = mx + c
where x=slope, y=intercept.

## Use Cases
- Salary prediction
- House price prediction
- Sales forecasting
