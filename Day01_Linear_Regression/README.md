## Definition
Linear Regression is a supervised learning algorithm used to
predict a continuous dependent variable based on input features.
It provides a best fit line between a one independent variable and one dependent variable.

## Formula
y = mx + c
where m=slope,c=intercept.

## Use Cases
- Salary prediction
- House price prediction
- Sales forecasting
